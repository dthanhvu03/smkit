# Decision Log

> **Vai trò:** Log quyết định đã chốt theo thời gian — agent ghi khi có decision mới.
> **Khác với:** `project-docs/decisions.md` (baseline kiến trúc, user điền).

## Format

```
### [YYYY-MM-DD] Decision Title

**Context:** Tại sao cần quyết định này
**Options considered:** Các lựa chọn đã xem xét
**Decision:** Chọn gì
**Rationale:** Tại sao chọn
**Consequences:** Hệ quả của quyết định
```
